传送类指令

/tp 实体传送指令（简化版）
▼

完整语法：

/tp <目标> <x> <y> <z> [旋转角度X] [旋转角度Y]
/tp <目标> <目标实体>
/tp <目标> <目标实体> facing <朝向实体>
/tp <目标> <目标实体> facing <x> <y> <z>

用途：将玩家或实体传送到指定坐标、目标实体，或朝向指定方向。

/teleport 高级传送指令（含安全传送）
▼

完整语法：

/teleport <目标> <x> <y> <z> [rx] [ry] [是否安全传送]
/teleport <目标> <目标实体> facing <朝向实体>
/teleport <目标> <目标实体> facing <x> <y> <z>

用途：相比/tp支持安全传送判断，防止玩家卡入方块。

游戏规则与世界指令

/gamerule 修改世界游戏规则
▼

完整语法：

/gamerule <规则名> [true/false/数值]

常用规则：

  • doDaylightCycle - 昼夜交替

  • keepInventory - 死亡保留物品
  • doMobSpawning - 生物生成
  • commandBlockOutput - 命令方块输出日志
  • showDeathMessages - 显示死亡信息

▼

/execute if <条件> run <指令>
/execute unless <条件> run <指令>
